

Atencion: If you use the webnotifications plugin you must disable it before start and reinstall after.

1 - In your GLPI make a copy of the folders css, pics and inc and the file index.php;


2 - Copy the folders css, fonts, pics and inc and the index.php file from GLPI Bootstrap folder to your server;


3 - To change the logo create your own image with size 200px x 200px, named to logo_big.png and copy to glpi/pics folder;


4 - To return to default GLPI remove the file index.php and the folders css, inc, pics and fonts and restore the folders backuped in step 1.


#### PT-BR

Atenção: se você usa o plugin Webnotifications você deve desabilitá-lo antes de começar e depois reinstalá-lo.

1 - Faça uma cópia das pastas css, inc e pics e do arquivo index.php do seu GLPI;

2 - Copie as pastas css, fonts, inc e pics e o arquivo index.php do arquivo descompactado para seu GLPI;

3 - Para mudar o logo crie sua imagem com tamanho 200px x 200px, com o nome logo_big.png e copie-a para a pasta glpi/pics;

4 - Para voltar ao GLPI padrão remova as pastas css, inc, fonts e pics e o arquivo index.php e restaure os arquivos do backup do passo 1.
