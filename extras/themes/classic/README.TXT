

1 - Make a copy of the folders css and pics and the file index.php from your GLPI;

2 - Copy the css and fonts folders, the index.php file of your files uncompressed for GLPI;

3 - Copy the files in folder pics to the pics folder in GLPI;

4 - To change the logo create your own image with size 200px x 200px, named to logo_big.png and copy to glpi/pics folder.


#### PT-BR


1 - Faça uma cópia das pastas css e pics e do arquivo index.php do seu GLPI;

2 - Copie as pastas css, fonts e pics e o arquivo index.php do arquivo descompactado para seu GLPI;

3 - Copie os arquivos da pasta pics para a pasta pics do GLPI;

4 - Para mudar o logo crie sua imagem com tamanho 200px x 200px, com o nome logo_big.png e copie-a para a pasta glpi/pics.
