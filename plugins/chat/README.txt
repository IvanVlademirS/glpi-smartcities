Plugin for Chat in GLPI, using livehelperchat.

INSTALL:

1 - Install the plugin; 

2 - In Setup -> Plugins click in plugin name in table to go to config page;

3 - Enable the plugin;

4 - Go to Plugin in top menu and click in chat to configure;

5 - In the first access will be configured the chat system. If you get errors:
chown -R www-data. <path_to_glpi>/plugins/chat   OR
chown -R wwwrun. <path_to_glpi>/plugins/chat

6 - Enter your information to access the mysql. The database name should be "lhc";

7 - Access the system and create departments and users;

8 - For more configuration details visit: http://livehelperchat.com/documentation-6c.html

####### PT_BR ############

INSTALAÇÃO:

1 - Instale normalmente o plugin, copiando para a pasta plugins no servidor;

2 - No menu Configurar -> Plugins clique no nome do plugin para abrir a página de configuração;

3 - Habilite o plugin;

4 - Clique em Plugin, no menu superior e selecione Chat;

5 - No primeiro acesso será configurado o sistema de chat. Se encontrar erros:
chown -R www-data. <path_to_glpi>/plugins/chat   OR
chown -R wwwrun. <path_to_glpi>/plugins/chat

6 - Informe os dados para acessar o mysql. O nome do banco deve ser "lhc";

7 - Acesse o sistema e crie os departamentos e usuários;

8 - Para maiores detalhes da configuração do chat acesse: http://livehelperchat.com/documentation-6c.html



UNINSTALL

1 - In Configuration -> Plugins click in plugin name to go to config page.

2 - Disable plugin.

3 - Uninstall plugin;
